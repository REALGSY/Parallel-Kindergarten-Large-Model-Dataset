# 平行幼儿园数据集 README

## 1. 数据集简介

**平行幼儿园数据集** 是一个面向幼儿教育、亲子陪伴与儿童友好型大语言模型训练的数据集。该数据集主要围绕幼儿园日常生活、儿童认知启蒙、情绪陪伴、故事互动、安全教育、习惯培养等场景构建，旨在为儿童友好型对话模型、幼儿教育助手、故事生成模型以及安全对齐模型提供训练与评测数据支持。

数据集内容尽量采用简单、温和、积极、易理解的语言风格，适合用于构建面向低龄儿童的智能交互系统。

---

## 2. 数据集目标

本数据集主要服务于以下目标：

1. **儿童友好对话能力训练**  
   让模型能够使用适合幼儿理解的语言进行回答，避免过于复杂、成人化或生硬的表达。

2. **幼儿教育场景适配**  
   支持模型在认知启蒙、生活习惯、情绪管理、简单科普、故事陪伴等场景下生成合适内容。

3. **安全对齐与风险规避**  
   帮助模型识别并避免不适合儿童的内容，例如暴力、恐吓、成人话题、危险行为指导等。

4. **亲子与教师辅助应用**  
   支持面向家长、幼儿教师和儿童教育产品的智能问答、内容生成和互动设计。

---

## 3. 适用场景

本数据集可用于以下任务：

- 儿童友好型大语言模型微调
- 幼儿教育对话系统训练
- 亲子陪伴聊天机器人
- 儿童故事生成与续写
- 幼儿情绪安抚与正向引导
- 儿童安全教育问答
- 幼儿园生活场景问答
- 模型安全对齐数据构建
- LoRA / QLoRA / SFT 微调实验
- DPO / RLHF 偏好数据构建的基础数据

---
